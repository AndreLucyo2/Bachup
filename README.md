# Bachup
Windows Backup Utility

This tool was created to make it easier to bachup data. 

### Features

* Zip Bachups
* Auto Saves
* Bachup History
* Theme Colors
* Dark/Light Theme


### Supported Backup File Types
* GDB
* TXT
* LAS
* SHP
